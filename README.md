## Особенности

- Поддерживаемые значения из вводного файла:
  - `point x y`
  - `rect x1 y1 x2 y2`
  - `hline x1 x2 y`
  - `vline y1 y2 x`
  - `circle x y r`
  - `dline x1 y1 x2 y2`
- Автоматическое масштабирование с сохранением пропорций (большая сторона по default — 1024 пикселя (может быть изменено)).
- Отрисовка в чёрно-белом цвете (можно задавать другие цвета для каждого объекта  и для фона).

## Сборка и запуск
### Рекомендуемая последовательность действий:

1. Создайте директорию для сборки проекта:
```bash
mkdir build && cd build
```
2. Из нее сконфигурировать проект:
```bash
cmake ../CMakeLists.txt
```
3. Собрать пример (example):
```bash
make example
```
4. Запустить
```bash
./example 
```
5. В выбранной папке (согласно данной инструкции **build**) появится папка **files** с исходными входными документами и результирующий bmp-файлами.
